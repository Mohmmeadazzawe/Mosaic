# خطة بناء موقع Mosaic HRD من الصفر

## نظرة عامة

إعادة بناء موقع https://mosaic-hrd.org/ باستخدام Next.js 14+ مع App Router. سيتم بناء جميع الصفحات ببيانات وهمية (Mock Data) واستخدام روابط الصور مباشرة من الموقع الأصلي، ثم ربط APIs لاحقاً.

## هيكل المشروع

```
Mozaic/
├── app/
│   ├── layout.tsx              # Layout الرئيسي
│   ├── page.tsx                 # الصفحة الرئيسية

│   │   └── page.tsx            # صفحة من نحن
│   ├── services/
│   │   ├── page.tsx            # صفحة الخدمات
│   │   └── [slug]/

│   ├── projects/
│   │   ├── page.tsx            # صفحة المشاريع
│   │   └── [slug]/
│   │       └── page.tsx       # صفحة تفاصيل المشروع
│   ├── news/
```

## الصفحات المطلوبة

### 1. الصفحة الرئيسية (`app/page.tsx`)

**المكونات:**

- Hero Section مع صورة رئيسية من الموقع الأصلي
- قسم الخدمات الرئيسية (3-6 خدمات)
- قسم المشاريع الأخيرة (آخر 3-6 مشاريع)
- قسم الأخبار الأخيرة (آخر 3-6 أخبار)
- قسم الإحصائيات (إن وجد)
- قسم الشركاء/الشعارات (إن وجد)

**روابط الصور:**

- استخدام روابط مباشرة من `https://mosaic-hrd.org/` للصور

### 2. صفحة من نحن (`app/about/page.tsx`)

**المحتوى:**

- نبذة عن المؤسسة
- الرؤية والرسالة
- القيم
- الفريق (إن وجد)
- الإنجازات/الجوائز (إن وجد)

### 3. صفحة الخدمات (`app/services/page.tsx`)

**المكونات:**

- قائمة بجميع الخدمات
- بطاقات الخدمات مع صور
- فلترة/تصنيفات (إن وجدت)

**صفحة تفاصيل الخدمة** (`app/services/[slug]/page.tsx`):

- تفاصيل الخدمة الكاملة
- صور توضيحية
- نموذج طلب الخدمة (إن وجد)

### 4. صفحة المشاريع (`app/projects/page.tsx`)

**المكونات:**

- شبكة/قائمة المشاريع
- فلترة حسب الفئة
- بحث (اختياري)
